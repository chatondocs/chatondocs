# ChatOnDocs

> Platform dokumentasi modern yang mudah digunakan dan powerful.

## 📖 Tentang

ChatOnDocs adalah platform dokumentasi yang dirancang untuk memudahkan tim dalam membuat, mengelola, dan berbagi dokumentasi secara kolaboratif.

## ✨ Fitur

- 📝 Editor markdown yang intuitif
- 🔄 Kolaborasi real-time
- 🎨 Template dokumentasi siap pakai
- 🔍 Pencarian cepat dan efisien
- 📱 Responsive design

## 🚀 Quick Start

### Prasyarat

- Node.js 18+
- npm atau yarn

### Instalasi

```bash
git clone https://github.com/chatondocs/chatondocs.git
cd chatondocs
npm install
npm run dev
```

### Penggunaan

1. Buka `http://localhost:3000`
2. Buat akun atau login
3. Mulai buat dokumentasi pertama Anda
