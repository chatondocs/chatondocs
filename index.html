Getting Started with ChatOnDocs

ChatOnDocs is a modern documentation platform designed to make it easy for teams to create, manage, and share documentation collaboratively. Follow this quickstart guide to get up and running in minutes.

Prerequisites

Before you begin, make sure you have the following installed:

  • Node.js 18+ — Download Node.js
  • npm or yarn — Package manager for JavaScript
  • Git — Version control system
Note

This guide assumes you're starting from scratch. If you have an existing project, you can skip the project creation step.

Create a New Project

Start by creating a new project directory and initializing it:

Terminal 
mkdir my-docs-site
cd my-docs-site
npm init -y

Install Dependencies

Install the ChatOnDocs core package:

Terminal 
npm install @chatondocs/core

Or with yarn:

Terminal 
yarn add @chatondocs/core

Configure Your Site

Create a configuration file at the root of your project:

chatondocs.config.js 
export default {
  title: 'My Documentation',
  description: 'A modern docs site built with ChatOnDocs',
  theme: {
    primaryColor: '#228be6',
    logo: '/logo.svg',
  },
  navigation: [
    { title: 'Getting Started', path: '/getting-started' },
    { title: 'API Reference', path: '/api' },
    { title: 'Guides', path: '/guides' },
  ],
}

Write Your First Document

Create your first documentation page in the docs/ directory:

docs/introduction.md 
# Introduction

## Welcome to ChatOnDocs

ChatOnDocs makes it easy to create beautiful documentation.

### Features

- Markdown support
- Dark mode out of the box
- Full-text search
- Responsive design

Start the Development Server

Run the development server to preview your documentation:

Terminal 
npx chatondocs dev

Your documentation site will be available at http://localhost:3000.
